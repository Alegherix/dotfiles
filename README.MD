# Personal Dotfiles
This is my setup of personal dotfiles to use whenever seting up a new system.

## System
**Os**: Arch
**DE**: I3
**Terminal**: Alacritty
**Filemanager**: Ranger
**Bar**: Polybar
**Editor**: VScode
**Discord**: BetterDiscord with Custom Theme
**Spotify**: Spicetify
**Browser**: Vivaldi
